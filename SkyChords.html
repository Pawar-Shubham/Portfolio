Intro

SkyChords is an android application which can generate an infinite number of chord progressions, or to put it another way, a straightforward melody. Through SkyChords, these chord progressions can be continuously developed.

SkyChords makes use of technology Native React for the front end and Python and MySQL as its Back-end technologies.

To create SkyChords, a variety of Python modules, including simpleaudio, numpy, scipy, wav, etc., were used.

Both the midi file and the wav file of the chord progression can be created by SkyChords.

The user of SkyChords has the option to save and rename their favourite midi and wav files.

The SkyChords-produced midi file can be used in a DAW, or digital audio workstation.
